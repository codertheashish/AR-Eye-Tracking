# 👁️ Eye Tracking AR Experience

A futuristic AI-powered Eye Tracking Augmented Reality experience built using MediaPipe Face Mesh, HTML5 Canvas, and JavaScript. The application tracks facial landmarks in real-time and generates immersive laser effects, particle systems, matrix animations, and dynamic audio feedback based on eye movement and face tracking.

## 🚀 How It Works

1. User grants webcam access.
2. MediaPipe Face Mesh detects facial landmarks.
3. Eye iris positions are tracked in real time.
4. Face movement controls dynamic visual effects.
5. Eye direction generates laser beam animations.
6. Audio reacts to movement intensity.
7. Matrix background and particles create an immersive AR experience.
